Los lentes fotografícos

Se pueden encontrar diferentes tipos de lentes, cada uno para escenarios específicos y características únicas, los primeros que se pueden describir son los lentes gran angulares, son lentes especializados para la fotografía de paisajes, dentro de este tipo se encuentran lentes como el lente 8-15mm, o el 10-22mm, por mencionar un par.

Luego se pueden encontrar los lentes fijos, que son lentes que mantienen una distancia focal fija, pero con mayor apertura del diafragma (sirve para mayor desenfoque del fondo), dentro de estos lentes podemos encontrar el 50mm.

Y finalmente se encuentran los teleobjetivos que funcionan para fotografiar largas distancias, esto se consigue gracias a su distancia focal del lente, este se utiliza en casos como de fotografías de animales y en eventos deportivos, algunos lentes teleobjetivos son el 55-210mm o el 600mm (lente fijo).
